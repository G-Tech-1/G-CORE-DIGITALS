# G-CORE OFFICIAL WEBSITE

## Overview

G-Core Official is a modern multi-service portfolio website showcasing different business divisions including fashion, grooming, technology, and creative studios.  

The website is designed with a clean UI, responsive layout, and structured navigation to present services and portfolio content professionally.
